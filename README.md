# ⚡ Aero G7 | Luxury E-bike Landing Page

A high-performance, immersive digital experience for the **Aero G7 Luxury E-bike**. This project focuses on "Zero-Bloat" architecture, hitting **100/100 Lighthouse scores** while maintaining complex GSAP-driven animations.

[Live Demo](https://abduljabbarmahar.github.io/Luxury-Motorcycle-Landing-Page/)

## 🚀 Performance Metrics (Daniyal OS Standards)
- **FCP:** <500ms
- **Lighthouse:** 100/100 (Performance, Accessibility, Best Practices, SEO)
- **Interactions:** 60fps buttery-smooth GSAP ScrollTrigger transitions.

## 🛠️ Tech Stack & Workflow
- **Engine:** [Vite](https://vitejs.dev/) (Lightning-fast HMR)
- **Language:** TypeScript + React
- **Styling:** Tailwind CSS v4 (Utilizing **OKLCH** color space for premium perceptual lighting)
- **Animation:** GSAP (ScrollTrigger, Text Splitting, Custom Bezier easing)
- **Architecture:** Modular "Senior Creative Dev" structure — No unnecessary backend, focused on raw UI/UX excellence.

## 🎨 Visual Logic
- **Identity:** High-contrast Dark Mode with Metallic Silver and Electric accents.
- **Glassmorphism:** Navigation and Card UI utilizing 24px backdrop-blur and 1px borders.
- **Typography:** Premium serif headings paired with high-tracking sans-serif body text for an editorial feel.
